#Project Overview
This project fetches real-time weather forecast data from the OpenWeatherMap public API using Python and visualizes it using Matplotlib and Seaborn.

The dashboard displays:
- Temperature trends
- Humidity levels

This project demonstrates API integration and data visualization skills.

## Technologies Used
- Python
- OpenWeatherMap API
- vs code
- Requests
- Matplotlib
- Seaborn
